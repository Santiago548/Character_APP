<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.
* []()install all gem files
```sh
$ bundle install
```
* []() migrate and seed the database
```sh
$ rails db:migrate
$ rails db:seed
```
* []() To Start the Server
```sh
$ rails s
```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Santiago548/NPC_Generator
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
